Popspotz

Description
Popspotz is an app that allows users to search, rate and add pop-ups.

Inspiration
This project came out of collaboration with another colleague. He and I are huge fans of popups but felt that there wasn't a great way to find them. Popspotz is an attempt to bridge this gap and help users in searching and sharing the popups in a more streamlined manner.

Technologies
Ruby on Rails, Postgres, Javascript, HTML5, CSS3, Google Maps API

What's next?
We are working on adding filtering and sorting of results as well as using user's current location to help find popups nearest to them!

FRUGALEATER

Description
Frugaleater is an app that helps users in finding cheap eats, specifically everything under $10, using the Locu API.

Inspiration
I am a huge foodie and the epitome of a frugal eater. As an avid user of Yelp and other such food-related apps, I always found it hard to search for a dish that was specifically $10 or less. Frugaleater is a search engine for finding such cheap eats.

Technologies
Ruby on Rails, Postgres, HTML5, CSS3, Javascript

What's next?
I am working on adding search by location functionality and making it mobile friendly.

Chatbuddy

Description
Chatbuddy is a simple chat app built using Meteor.

Inspiration
This app was built as part of my research project at General Assembly. I was really keen on learning how to add chat functionality to any app. After some research, it seemed like Meteor would be ideal for trying that . So I kept it really simple and basic.

Technologies
Meteorjs

What's next?
I would like to allow users to add friends on github and actually chat with other developers they know rather than complete strangers.
